# 一.总体概览
在本科期间我们一共有三次专业实习,分别是
1. 认知实习 : 大一升大二的暑假,长春周边+校内实习
2. 兴城实习 : 大二升大三的暑假, 搬去朝阳之后就会去辽宁兴城,校外实习
3. 专业实习 : 大三升大四的暑假,校内实习

# 二.具体介绍
## 1.认知实习
**时间** : 大一升大二的暑假; 

**带队老师** : 路兴昌  张艳红  孟治国  李飞龙(学长, 就职于长光卫星)
## 2.兴城实习
**时间** : 大二升大三的暑假, 大约是44天, 从7.1-8.16(具体放假时间要看安排); 放假会一直持续到国庆之后, 国庆后返校即可

**带队老师** : 蒋立军  潘军

**实习内容**

**实习注意事项**
1. 看公众号的要求, 一定要提前准备好**物资**,兴城实习生活体验真的不好(海边 + 无空调 + 蚊虫多 + 洗澡不方便)
2. 会有大概1-2周的野外实习, 野外实习会很累, 做好心理准备, 备好防晒物资(长裤是必须的)
3. 实习总体体验**很水**, 且任务较简单(除了最后小组整体的提交的作业)

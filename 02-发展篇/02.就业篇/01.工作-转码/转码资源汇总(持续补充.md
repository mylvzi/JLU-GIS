# 技术学习与求职资源汇总

> 本文档用于整理在技术学习、系统基础、求职准备过程中长期使用和认可的优质资源，作为索引与备忘，不追求全面，只保留高质量来源。

## 一、技术栈学习

### C 语言

* **鹏哥 C 语言（Bilibili）**
  [https://www.bilibili.com/video/BV1Vm4y1r7jY/](https://www.bilibili.com/video/BV1Vm4y1r7jY/)
  体系完整、讲解深入，对 C 语言底层理解帮助很大，适合系统学习。

### Linux

* **韩顺平 Linux 教程（Bilibili）**
* **黑马程序员 Linux 系列（Bilibili）**
  覆盖 Linux 基础使用、常见命令、系统概念，适合作为入门与补充。

### 计算机基础自学体系

* **CS 自学指南（csdiy）**
  [https://csdiy.wiki/#_9](https://csdiy.wiki/#_9)
  结构化的计算机自学路线整理，被广泛认可，适合作为长期学习参考框架。

## 四、技术社区与信息源

* **Stack Overflow**
  [https://stackoverflow.com/questions](https://stackoverflow.com/questions)
  高质量英文技术问答社区，适合定位具体问题和理解底层原理。

* **阮一峰的网络日志 · 科技周报**
  [https://www.ruanyifeng.com/blog/weekly/](https://www.ruanyifeng.com/blog/weekly/)
  用于了解技术趋势、工具更新与行业动态。

* **Reddit**
  [https://www.reddit.com/](https://www.reddit.com/)
  覆盖工程实践、技术讨论与前沿话题的国际社区。

### 1. 什么是“三方协议”？

全称《普通高等学校毕业生就业协议书》。

简单来说，它是你、学校、用人单位三方之间签订的一份正式约定。其核心作用是阐明：毕业生愿意去该单位工作，单位愿意接收该毕业生，学校负责办理报到证等后续档案转递手续。

- **有效期：** 从签约起，到你正式入职并签订《劳动合同》为止。
    
- **法律属性：** 属于民事合同（受《民法典》约束），不属于《劳动法》范畴。
    

你吉的三方目前是采用**网签**的形式进行签约, 如果有什么不清楚的可以咨询导员

---

### 2. 三方协议 vs 劳动合同：有什么区别？

| **维度**   | **三方协议**      | **劳动合同**        |
| -------- | ------------- | --------------- |
| **签署主体** | 毕业生、学校、企业（三方） | 劳动者、企业（二方）      |
| **主要目的** | 确立就业意向，处理档案迁徙 | 确立劳动关系，规定岗位权利义务 |
| **法律依据** | 《民法典》         | 《劳动法》、《劳动合同法》   |
| **时效**   | 报到入职后即失效      | 入职即生效，长期有效      |

### 3. 签约前必看

很多同学签三方时只看公司名，其实**补充协议（备注栏**才是重头戏。请务必确认以下内容：

- **薪资构成：** 不要只听“总包”，要写清基本工资、绩效、年终奖以及**五险一金**的缴纳基数[[关于五险一金]]。
    
- **工作地点：** 明确是总部还是分支机构，防止被“全国随机派驻”。
    
- **违约金：** 这是最关键的一点。法律规定三方协议可以设置违约金，通常在 3000–8000 元不等。如果数额过高（如几万元），一定要谨慎！(一般来说,偏国企性质的企业违约金都比较高,私企会相对低一些)
    
- **备注条款：** 比如承诺的解决落户、提供宿舍、岗位实习期要求等，口头承诺无效，写进备注才靠谱。
    
* **违约时间** : 很多企业会设置一个**统一的违约时间**, 来批量处理三方违约(实际上, 可以通过一些手段, 提前解约, 可自行上网查询, 这里不做介绍)
* **签约主体** : 这也是需要注意的一点, 即落款公司名称, 避免出现**面试公司与实际入职公司不匹**的现象

---

### 4. 关于“违约”的温馨提示

如果你签了 A 公司，后来拿到了更好的 B 公司 Offer，想要毁约：

1. **流程：** 你需要向 A 公司申请解约 → 拿到 A 公司的**解约函** → 交给学校换领新的三方协议 → 签 B 公司。
    
2. **代价：** 通常需要支付协议约定的违约金。
    
3. **建议：** 毁约要趁早，不要拖到快入职才告知企业，给对方留出重新招聘的时间。


### 一、 什么是 Markdown？

简单来说，Markdown 是一种“用键盘就能完成排版”**的纯文本写作方式。

在 Word 里，你想把文字变粗，得选中文本，再点一下工具栏的“B”。

在 Markdown 里，你只需要在文字两边加两个星号：**我要变粗**。

**它的核心逻辑是：让你专注于写作本身，而不是纠结鼠标点哪儿排版。**

---

### 二、 为什么建议你一定要学会它？

1. **极速排版，手不离键盘**：写标题、加粗、列清单，全部通过简单的符号完成，你的思路不会被鼠标点击打断。
    
2. **全平台通用，永不乱码**：MD 文件本质是纯文本（.md后缀），用记事本都能打开。无论是发给老师还是同步到手机，排版永远保持原样。
    
3. **对代码和公式极度友好**：
    
    - **理工科福利**：它支持 LaTeX 公式。
        
    - **计算机福利**：插入代码块自带高亮，非常清晰。
        
4. **导出方便**：你可以一键把 MD 笔记转成精美的 PDF 或网页（HTML）。

### 三、 5 分钟就能学会的常用语法

#### 1. 标题（井号后面加空格）

Markdown

```
# 一级标题（最粗最大的）
## 二级标题
### 三级标题
```

#### 2. 加粗与倾斜

Markdown

```
**这是加粗**
*这是斜体*
```

#### 3. 列表

Markdown

```
* 不用点工具栏
* 直接用星号或减号加空格
* 它就自动变成圆点列表了
```

#### 4. 插入代码

使用反引号（键盘 Esc 下面那个键）：

Markdown

```
`printf("Hello World");`
```

#### 5. 引用（引用名言或说明）

Markdown

```
> 这是一段引用的文字，前面加个大于号就行。
```

更多语法可以查看[Markdown 基本语法 | Markdown 教程](https://markdown.com.cn/basic-syntax/)
